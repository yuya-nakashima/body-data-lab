# Body Data Lab Overview

## 目的

Body Data Lab は、身体データを小さく集めて保存し、あとから分析や可視化につなげるための実験・開発基盤です。

## 想定するデータの流れ

Android / Watch / その他入力元 -> API -> SQLite -> ETL -> Dashboard

この流れは現時点では仮置きであり、最初の実装を進めながら更新します。

## まず作る候補

- 最小限のデータ取り込み API
- 生データと正規化データを置くための SQLite スキーマ
- 取り込んだデータを整形するシンプルな ETL
- データ到達確認のための簡易ダッシュボード
