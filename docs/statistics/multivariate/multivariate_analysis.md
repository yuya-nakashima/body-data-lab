# 多変量解析

## 概要

多変量解析は、複数の変数を同時に扱って構造や差異を捉える総称です。Body Data Lab では、単一指標では見えにくいコンディションや生活パターンを整理する中核になります。

## 主要概念

- 多次元データの要約
- 相関構造と共分散構造
- 次元削減
- 分類とクラスタリング
- 潜在構造の推定

## 重要な数式

- 平均ベクトル: `\mu = E[X]`
- 共分散行列: `\Sigma = E[(X-\mu)(X-\mu)^\top]`
- マハラノビス距離: `d(x,\mu)^2 = (x-\mu)^\top \Sigma^{-1}(x-\mu)`

## Body Data Lab での利用例

- 複数の身体指標からコンディションの潜在構造を探る
- 通常日と疲労日の分布差を多次元空間で把握する
- 高度分析に進む前に、変数同士のまとまりを確認する

## 未理解でも先に進めるポイント

- まずは相関行列と散布図行列を見るだけでも、多変量解析の入口になります。
- すべての手法を同時に学ぶ必要はなく、PCA とクラスタ分析から始めるのが現実的です。

## 今後深掘りすべき論点

- 小標本で多変量手法を使うときの不安定性
- 欠測値が多い場合の前処理
- 説明可能性と予測精度の優先順位
