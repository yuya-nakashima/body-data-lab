# 用語集

## 概要

この用語集は、統計学とアプリ設計の会話で頻出する語を揃えるための簡易辞書です。厳密定義より、誤解を減らすことを重視します。

## 主要概念

| 用語 | 概要 |
| --- | --- |
| 記述統計 | データを要約する方法 |
| 推測統計 | 母集団や将来値について推論する方法 |
| 目的変数 | 説明したい対象 |
| 説明変数 | 目的変数の変化を説明する候補 |
| 共線性 | 説明変数同士が強く似ている状態 |
| 定常性 | 時系列の統計的性質が時間で大きく変わらないこと |
| 潜在変数 | 直接は観測しないが背後にあると考える量 |
| 事後分布 | データ観測後のパラメータ分布 |

## 重要な数式

- このファイルでは数式より概念の統一を優先します。
- 詳細な式は各分野のファイルを参照します。
