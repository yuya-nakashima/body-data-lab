# 記号一覧

## 概要

このファイルは、統計ドキュメント内で繰り返し使う記号をざっくり統一するための補助資料です。厳密な流派差はありますが、Body Data Lab 内で読みやすさを優先します。

## 主要概念

- 母集団と標本の記号を分ける
- スカラー、小文字ベクトル、行列の区別を意識する
- 推定値にはハットを付ける

## 重要な数式

- 平均: 母平均 `\mu`, 標本平均 `\bar{x}`
- 分散: 母分散 `\sigma^2`, 標本分散 `s^2`
- 回帰係数: 真値 `\beta`, 推定値 `\hat{\beta}`
- 確率変数 `X`, 実現値 `x`

## 未理解でも先に進めるポイント

- すべて暗記する必要はなく、必要なときに参照できれば十分です。
- 数式が難しいときは、まず「真値か推定値か」だけ区別できれば進めます。
