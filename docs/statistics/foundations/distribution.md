# 分布

## 概要

分布は、データがどの値をどの程度取りやすいかを表します。統計モデルの選定では「何を正規分布とみなすか」「カウントデータとして扱うか」が分析品質を左右します。

## 主要概念

- 離散分布と連続分布
- 確率質量関数、確率密度関数、分布関数
- 正規分布、二項分布、ポアソン分布、指数分布
- 多変量正規分布
- 分布の近似と変換

## 重要な数式

- 正規分布: `X \sim N(\mu, \sigma^2)`
- 二項分布: `P(X=x) = {n \choose x} p^x (1-p)^{n-x}`
- ポアソン分布: `P(X=x) = e^{-\lambda} \lambda^x / x!`
- 標準化: `Z = (X - \mu) / \sigma`

## Body Data Lab での利用例

- 歩数はカウントデータとしてポアソン系モデル候補を考える
- 睡眠時間や心拍は連続量として正規近似の妥当性を確認する
- 複数指標を同時に扱うとき、多変量正規分布の発想が PCA や判別分析につながる

- まずは「歩数は整数」「睡眠時間は連続値」という型の違いを意識できれば十分です。
