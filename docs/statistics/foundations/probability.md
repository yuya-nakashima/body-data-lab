# 確率

## 概要

確率は、不確実な事象を定量化するための言語です。Body Data Lab では「今日の疲労感が高い確率」「心拍が通常より高い状態が続く確率」といった不確実性の整理に直結します。

## 主要概念

- 標本空間、事象、確率測度
- 条件付き確率と独立性
- ベイズの定理
- 確率変数、期待値、分散、共分散
- 大数の法則と中心極限定理の直感

## 重要な数式

- 条件付き確率: `P(A|B) = P(A \cap B) / P(B)`
- 独立性: `P(A \cap B) = P(A) P(B)`
- 期待値: `E[X] = \sum_x x P(X=x)` または `E[X] = \int x f(x) dx`
- 分散: `Var(X) = E[(X - E[X])^2]`
- 共分散: `Cov(X, Y) = E[(X - E[X])(Y - E[Y])]`
- ベイズの定理: `P(A|B) = P(B|A) P(A) / P(B)`

## 未理解でも先に進めるポイント

- MVP 段階では、確率の厳密体系より「不確実性がある」という認識だけでも十分役立ちます。
- まずは条件付き確率と期待値の意味が分かれば、分析設計の会話は進めやすいです。

## 今後深掘りすべき論点

- 個人データで独立性を仮定してよい場面の見極め
- 条件付き確率と因果解釈の混同回避
- 欠測メカニズムを確率モデルとして扱う方法
