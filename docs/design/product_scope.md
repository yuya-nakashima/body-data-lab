# Product Scope

## MVP でやること

- Android 中心で身体データを取り込む
- 主要データを保存し、日次・週次の基本集計を行う
- 歩数、睡眠、心拍、稽古有無、手動メモを最低限扱えるようにする
- 折れ線グラフ、平均、移動平均などの基礎可視化を提供する
- データ品質の確認と欠損把握ができるようにする

## 今はやらないこと

- iPhone 最適化前提の設計
- 本格的なソーシャル共有
- 汎用的な医療診断支援
- 高負荷なリアルタイム推論基盤
- 複雑な実験計画 UI の初期実装

## 将来拡張

- 多変量解析によるコンディション指標生成
- 時系列予測や状態遷移モデル
- ベイズ推定による個人適応型分析
- サーバー側バッチ分析と長期履歴管理
- ストレス、体重、食事、運動内容の追加

## スコープ境界

- 健康改善の補助は行うが、医療判断は目的にしない
- 自己研究と学習用途を重視するが、一般向け展開は後で判断する
- 現時点では分析の正確性よりも、段階的に検証可能な構造を優先する
